Conway's Game of life

Being inspired by my school project from CS111, I implemented the game again in JavaScript and used HTML5 canvas to display the cells and animation (browsers supported HTML5 required).

From wikipedia: The Game of Life, also known simply as Life, is a cellular automaton devised by the British mathematician John Horton Conway in 1970.

Rules:
1.Any live cell with fewer than two live neighbours dies, as if caused by under-population.
2.Any live cell with two or three live neighbours lives on to the next generation.
3.Any live cell with more than three live neighbours dies, as if by overcrowding.
4.Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

Supported browsers:
  • Firefox 4+
  • Google Chrome
  • Opera 10.6+
  • Apple Safari 4+
